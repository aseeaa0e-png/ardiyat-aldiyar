# 🏗️ دليل استخدام القالب — معرض أرضيات جديد

هذا الموقع **ملف واحد** (`index.html`). لإطلاق معرض أرضيات جديد، انسخ المشروع وعدّل النقاط التالية فقط. كل التعديلات داخل `index.html` إلا الملفات (الفيديو/الصور).

---

## ⚡ الطريقة الأسرع: مستودع قالب (Template Repository)

1. على GitHub افتح **Settings** للمستودع `ardiyat-aldiyar`.
2. فعّل خيار **Template repository** (علامة ✓).
3. لكل عميل جديد: اضغط **Use this template → Create a new repository** — يطلع لك نسخة كاملة جاهزة باسم جديد.
4. فعّل **GitHub Pages** للنسخة الجديدة (Settings → Pages → Branch: main).

بهذا تحصل على موقع مستقل لكل عميل بدون نسخ يدوي.

---

## ✅ قائمة التعديل لكل معرض جديد

### 1) ⚙️ كتلة الإعدادات `CONFIG` (الأهم — مكان واحد)
في أعلى السكربت مباشرة تجد `const CONFIG = { ... }`. عدّل منها كل شيء أساسي:
- `whatsapp` — رقم الواتساب بصيغة `9665xxxxxxxx`
- `accent` — اللون الرئيسي: `نحاسي` | `زيتوني` | `طيني`
- `rating` و `reviews` — التقييم وعدد المراجعات
- `brand` — اسم المعرض (عربي/إنجليزي)
- `address` — العنوان
- `hours` — ساعات العمل
- `footer` — جملة التذييل

هذه الكتلة وحدها تغطّي الاسم، التواصل، اللون، والعنوان لكل الموقع تلقائياً.

### 2) المنتجات والأسعار
ابحث عن `const PRODUCTS`. كل سطر = منتج فيه:
- `ar`/`en` الاسم، `cat` الفئة (parquet/ceramic/porcelain/spc)
- `price` سعر المتر، `install` التركيب
- `color` اللون التقريبي، `tx` نوع النسيج (wood/marble/stone/clay/ceramic/spc)
أضف/احذف منتجات بحرية. النُسج تتولّد تلقائياً بالكود.

### 3) الأسئلة الشائعة (الشات بوت)
ابحث عن `const FAQ` — عدّل الأجوبة (أسعار، تركيب، ضمان، العنوان...) لتطابق المعرض.

### 4) آراء العملاء
ابحث عن `const TESTI` — بدّل الشهادات بأسماء عملاء المعرض.

### 5) الفيديو والصور
استبدل الملفات بنفس الأسماء:
- `intro.mp4` — فيديو المقدمة
- `cover.jpg` — غلاف الفيديو (يظهر قبل التشغيل) — يفضّل بدون كتابة
- `poster.jpg` — صورة المشاركة (واتساب/تويتر) — يفضّل فيها اسم المعرض

### 6) ميتا المشاركة وعنوان الصفحة
في أعلى الملف داخل `<head>`: غيّر `<title>` ووسوم `og:title` / `og:description`.

## 🎯 ملخص: 6 نقاط فقط
`CONFIG` (الاسم/الواتساب/اللون/العنوان) · `PRODUCTS` (الأسعار) · `FAQ` · `TESTI` · ملفات الوسائط · `<head>`

كل شيء آخر (AR، المستشار، الحاسبة، المقارنة، الشات، تبديل اللغة، توليد النُسج) يشتغل تلقائياً بلا تعديل.
