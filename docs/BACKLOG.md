## المخاطر الحالية (Risks)

| الخطر | الأثر | التخفيف |
|------|------|---------|
| **شعارات تابي/تمارا غير رسمية** | قانوني/مصداقية عند الإطلاق التجاري | لا تُطلق تجاريًا قبل الاتفاق واستبدال الشعارات. |
| **الكاميرا الحية تعتمد على إذن الجهاز/HTTPS** | قد تفشل على بعض الأجهزة/المتصفحات | يوجد بديل «ارفع صورتك» يعمل دائمًا + رسالة عند الرفض. |
| **التوكن في سجلّ الأوامر** | تم استخدام GitHub Token أثناء الإعداد | يُنصح بإبطال التوكن وإنشاء غيره بعد التسليم. |
| **ملف واحد كبير (1146 سطر)** | صعوبة صيانة لمطوّر جديد | موثّق بالكامل في docs/ + كتلة CONFIG تعزل التعديلات الشائعة. |
| **بيانات نموذجية** | قد تُعرض بالخطأ كأنها حقيقية | تُستبدل قبل أي عرض فعلي (موثّق). |
| **النُسج إجرائية تقريبية** | ليست صور مواد حقيقية | كافية للنموذج؛ يمكن استبدالها بصور حقيقية لاحقًا. |
