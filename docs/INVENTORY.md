## ب) شرح الملفات الرئيسية

| الملف | الوظيفة |
|------|---------|
| `index.html` | الموقع بالكامل: `<head>` (خطوط + ميتا مشاركة + favicon)، CSS (حركات + أساسيات)، JavaScript (CONFIG، الحالة، البيانات، توليد النُسج، المنطق، دوال العرض). |
| `intro.mp4` | يُعرض في مقدمة الصفحة، مرة واحدة بلا تكرار. |
| `cover.jpg` | يظهر مكان الفيديو قبل تحميله (poster) — بدون نص. |
| `poster.jpg` | يظهر كبطاقة معاينة عند مشاركة الرابط (og:image / twitter). |
| `TEMPLATE-GUIDE.md` | خطوات وقائمة تعديل لإنشاء معرض جديد من القالب. |
| `project/*` | مخرجات Claude Design الأصلية — للمرجعية والمقارنة، لا يستخدمها الموقع الحيّ. |

## ج) الأقسام (Sections) داخل الصفحة الواحدة

ترتيب الظهور من أعلى لأسفل (كلها في `index.html`):

| # | الدالة | القسم | الوظيفة |
|---|--------|-------|---------|
| 1 | `renderHeader` | الهيدر | الشعار + تبديل اللغة + زر واتساب (ثابت أعلى الصفحة). |
| 2 | `renderVideoHero` | الفيديو | فيديو المقدمة + اسم المعرض + زرّا CTA. |
| 3 | `renderOfferBar` | شريط العرض | عرض «معاينة مجانية» نحاسي + زر حجز. |
| 4 | `renderAR` | جرّبها بمساحتك | رفع صورة/كاميرا حية + سحب خط + اختيار أرضية + حفظ. |
| 5 | `renderAdvisor` | المستشار الذكي | ٤ أسئلة → أفضل ٣ أرضيات بنسبة توافق وسبب. |
| 6 | `renderColorDetector` | كاشف اللون | رفع صورة جدار → اللون السائد → أقرب أرضية. |
| 7 | `renderCatalog` | الكتالوج | ١٢ منتج/٤ فئات + فلتر + تكبير + روابط حساب/مقارنة. |
| 8 | `renderCalculator` | الحاسبة | حساب التكلفة + تقسيط تابي/تمارا + تحميل عرض سعر. |
| 9 | `renderCompare` | المقارنة | مقارنة فئات + مقارنة مخصصة لمنتجين + خلاصة. |
| 10 | `renderWhy` | ليش الديار | ٤ مزايا (توصيل/ضمان/رد فوري/أسعار صادقة). |
| 11 | `renderTestimonials` | آراء العملاء | ٣ شهادات مع أسهم تمرير. |
| 12 | `renderOffer` | العرض المحدود | عرض كامل + سعر مشطوب + زر حجز. |
| 13 | `renderContact` | التواصل | بطاقة المقاولين + العنوان/الساعات/التقييم + واتساب. |
| — | `renderChatFAB` + `renderChat` | الشات بوت | زر عائم + نافذة دردشة (FAQ + تحويل واتساب). |
| — | `renderZoom` | نافذة التكبير | عرض مقرّب لأي خامة (Modal). |
| — | `renderBnpl` | شارة التقسيط | تُستدعى داخل الحاسبة (تابي/تمارا). |

## د) كتل البيانات (للتعديل لكل معرض)
- `CONFIG` — إعدادات المعرض (الاسم/الواتساب/اللون/العنوان/التقسيط).
- `PRODUCTS` — ١٢ منتجًا (اسم، فئة، سعر، تركيب، لون، نوع نسيج، خصائص).
- `STR` — جداول النصوص (عربي/إنجليزي).
- `ADVISOR_Q` — أسئلة المستشار الأربعة.
- `FAQ` — مواضيع الشات بوت (~١٣).
- `WHY` / `TESTI` — المزايا والشهادات.
